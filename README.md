# Doublon
Remove duplicate lines in a file. Warning, it will erase the previous file.

## Install

```
git clone https://github.com/rap2hpoutre/doublon.git
cargo build --release
mv target/release/doublon /usr/local/bin/doublon

doublon -h
```

## Usage

```
Doublon
Dedup lines in file.

Usage:
  doublon <file>
  doublon (-h | --help)

Options:
  -h --help     Show this screen.
```
